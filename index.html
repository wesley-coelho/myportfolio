Sou bacharel em engenharia elétrica com ênfase em eletrônica, estudante de Ciência de Dados e autodidata em linguagens de programação como C, C++, C#, Java, Javascript, Dart e Python. Durante o curso de engenharia elétrica que percebi que o que mais gostava era codificar. Na faculdade tive contato com a linguagem Assembly e C. Fiz meu TCC, um projeto de estufa controlada por microcontroladores PIC, utilizando Assembly, Nessa época lidei por 1 ano com programação e depuração de erros em Assembly.

Cheguei a cursar 2 anos de Análise e Desenvolvimento de Sistemas na Fatec de Ribeirão Preto, onde tive acesso a disciplinas de modelagem de banco de dados relacional, SQL, programação para dispositivos móveis e algoritmos. Na Fatec trabalhei com o framework Flutter, linguagem Dart, utilizei prototipagem de software e engenharia de requisitos para construir um aplicativo Android para venda de produtos. E também tive contato, pela primeira vez, com orientação a objetos, em C#.

Profissionalmente, desde 2007, tenho trabalhado para uma empresa de radiodifusão com conteúdo audiovisual, onde, sempre que possível, utilizo Javascript para automatizar tarefas do dia a dia criando scripts para ler dados de documentos de texto e planilhas, manipular arquivos na rede e consumir api externa.

Paralelamente, ao meu emprego formal, trabalho como programador autônomo, fazendo websites para amigos e clientes, soluções desktop e mobile. No meu tempo livre, estudo programação, aplicando em jogos, ou ferramentas para uso próprio.

Agora, estou buscando uma transição de carreira. Por isso, ingressei no curso de Ciência de Dados na Univesp EAD onde estou aprendendo a programar também em Python, aprofundando meus conhecimentos em estruturas de dados usando C++ e ampliando meus conhecimentos em estatística.

Considero-me uma pessoa resiliente, comprometida, bem educada e de fácil relacionamento. Cada aplicação que desenvolvo, por mais simples que seja, sinto-me muito realizado.
